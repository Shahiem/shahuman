# ShaHuman

Create your ideal character made with JavaScript.

## Building and running on localhost

### Install node modules
First install dependencies:

```sh
npm install
```

### Development

To create a development build:

```sh
npm run start
```

### Production

To create a production build:

```sh
npm run build
```
